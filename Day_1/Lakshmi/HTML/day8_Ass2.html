The Meta tag gives information about the data.It is in the head tag.It is not displayed on the page but it is used by browsers about how to display content or reload the page. The meta tag have different attributes .They are
  • charset:charset specifies the character encoding for webpage .By default we use UTF-8
  • Viewport:Viewport is the visible area of the webpage.It varies with the device [ width=device-width part is specifies width of the page is equal to device width., initial-scale=1.0 part specifies that the Initial zoom level of webpage.]
  • name:It specifies a name for the metadata like description,author,keyword,viewport
  • http-equiv: It provides an HTTP header for the information/value of content attribute.refresh value.
  • content : It specifies the value associated with the http-equiv or name attribute.
